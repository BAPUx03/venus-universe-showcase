## 🚀 Getting Started

```bash
# Clone
git clone https://github.com/<your-org>/venus-universe.git
cd venus-universe

# Install (fast, deterministic)
bun install

# Dev server
bun run dev
# → http://localhost:8080

# Production build
bun run build
```

### 🔐 Required Environment Variables

Auto-provisioned by Lovable Cloud, no manual setup needed:

```env
SUPABASE_URL=…
SUPABASE_PUBLISHABLE_KEY=…
LOVABLE_API_KEY=…           # AI gateway + connectors
SEMRUSH_API_KEY=…           # via Semrush connector
BREVO_API_KEY=…             # transactional email
TWOFACTOR_API_KEY=…         # OTP verification
```

## 📁 Project Structure

```text
src/
├── routes/                 # File-based routing (TanStack Start)
│   ├── __root.tsx          # Sitewide shell + head defaults
│   ├── index.tsx           # Landing page
│   ├── eoi.tsx             # Expression of interest form
│   ├── studio.tsx          # Admin panel + SEO dashboard
│   └── api/                # Server routes (webhooks, MCP)
├── components/
│   ├── site/               # Hero, Gallery, StickyBar, EoiPopup…
│   └── ui/                 # shadcn primitives
├── lib/
│   ├── mcp/                # MCP server tools
│   └── seo/                # Semrush + Auto-SEO agent server fns
├── content/                # Editable site content defaults
├── hooks/                  # useSiteContent, useLeadCapture…
└── integrations/supabase/  # Auto-generated client
```
